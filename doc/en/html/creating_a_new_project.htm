Creating a New Project

To create a new project:

1.Select File >> New or click on the Standard Toolbar.

2.You will be prompted to save the existing project (if changes were made to it) before the new project is created.

3.A new, unnamed project is created with all options set to their default values.

A new project is automatically created whenever EPANET first begins.

Note: If you are going to use a backdrop map with automatic pipe length calculation, then it is recommended that you set the map dimensions immediately after creating the new project (see Setting the Map's Dimensions).
