Source Quality Editor

The Source Quality Editor is a pop-up dialog box used to describe the quality of source flow entering the network at a specific node. This source might represent the main treatment works, a well-head or satellite treatment facility, or an unwanted contaminant intrusion. The dialog box contains the following fields:

Field Description
Source Type Select either:

·Concentration

·Mass Booster

·Flow Paced Booster

·Setpoint Booster

See Water Quality Sources for details.
Source Quality Baseline or average concentration (or mass flow rate) of source - leave blank to remove the source
Quality Pattern ID label of time pattern used to make source quality vary with time - leave blank if not applicable
