Graph Selection Dialog Box

The Graph Selection dialog box is used to select a type of graph and its contents to display. The choices available in the dialog box consist of the following:

ChoiceDescription

Graph TypeSelect a graph type

ParameterSelect a parameter to graph

Time PeriodSelect a time period to graph (does not apply to Time Series or System Flow plots)

Object TypeSelect either Nodes or Links (only Nodes can be graphed on Profile and Contour plots)

Items to GraphSelect items to graph (applies only to Time Series and Profile plots) -- see Selecting Items to Graph
