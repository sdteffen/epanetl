Opening an Existing Project

To open an existing project stored on disk:

1.Either select File >> Open or click on the Standard toolbar.

2.You will be prompted to save the current project (if changes were made to it).

3.Select the file to open from the standard Open File dialog box.

You can choose to open a file type saved previously as an EPANET project (typically with a .NET extension) or exported as a text file (typically with a .INP extension). EPANET recognizes file types by their content, not their names.
