What's New in Releases 2.00.11-12

·Object ID names can now contain up to 31 characters.

·The logic for determining the status of PRVs and PSVs was changed to produce more robust solutions, especially for PRVs in parallel.

·The parameters that control the convergence behavior of a hydraulic solution were made editable. See the description of CHECKFREQ, MAXCHECK, and DAMPLIMIT under Hydraulics Options.

·An improved computational scheme was introduced for computing water quality changes in storage tanks.

·A user's program settings are now stored in the user's Application\Epanet folder on their file system rather than to the Epanet installation folder.
