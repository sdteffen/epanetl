To display/hide the node legend:

select/unselect View >> Legends >> Node

or-

right-click on an empty portion of the Network Map and click Node Legend from the popup menu.

To display/hide the link legend:

follow the above procedure, except replace Node with Link.

To display/hide the time legend:

select/unselect View >> Legends >> Time

-or-

right-click on an empty portion of the Network Map and select/unselect Time Legend from the popup menu.

Any of these legends can also be hidden by double-clicking on them.
