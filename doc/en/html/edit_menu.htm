Edit Menu

The Edit Menu contains commands for editing and copying.

CommandDescription

Copy ToCopies the currently active view (map, report, graph or table) to the clipboard or to file

Select ObjectAllows selection of an object on the map

Select VertexAllows selection of link vertices on the map

Select RegionAllows user to outline a region on the map for selecting multiple objects

Select AllMakes the outlined region the entire viewable area of the map

Group EditEdits a property for the group of objects that fall within the outlined region of the map
