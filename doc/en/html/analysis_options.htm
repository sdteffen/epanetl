Analysis Options

Analysis Options control the way in which a network is analyzed. There are five different categories of options that can be selected from the Data Browser after choosing Options from the dropdown list (or by selecting Project >> Options on the menu bar):

· Hydraulics Options

·Quality Options

·Reactions Options

·Time Options

·Energy Options

To edit an option category, first select it in the Data Browser and then either:

· click the Edit button on the Browser

·double-click the option category

·press the Enter key
