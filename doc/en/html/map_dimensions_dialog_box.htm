Map Dimensions Dialog Box

The Map Dimensions dialog box is used to change the dimensions of the Network Map.

ItemDescription

Lower Left CoordinatesThe X,Y coordinates of the lower left point on the map

Upper Right CoordinatesThe X,Y coordinates of the upper right point on the map

Map UnitsUnits used to measure distances on the map. The choices are:

·Feet

·Meters

·Degrees

·None

Auto-Size ButtonClick to have dimensions be computed based on the coordinates of objects currently included in the map
