Editing an Object

To edit an object appearing on the map

click the Select Object button if its not already depressed

if the Property Editor is already visible then simply click on the object

if the Property Editor is not currently visible then either

- double-click the object on the map or

- right-click the object on the map and select Properties from the pop-up menu or

- click on the object and then click the Edit button on the Data Browser

To edit an object appearing in the Browser

select the object from the Data Browser and then either

- click the Browser's Edit button or

- double-click the item in the Browser or

- press the Enter key.
