Pattern Editor

The Pattern Editor edits the properties of a time pattern object. To use the Pattern Editor:

1.Enter a pattern ID label of 15 or less characters in the Pattern ID field.

2.Enter an optional description of the pattern in the Description field.

3.Enter multipliers for each time period of the pattern. The length of the time period is supplied as one of the project's Time Options. Its current value is displayed on the preview chart. To add more time periods than are currently showing, hit the Enter key when in the last cell of the editor.

4.As multipliers are entered the preview chart is redrawn to provide a visual depiction of the pattern.

5.Click the OK button to accept the pattern.

You can also click the Load button to load in pattern data that was previously saved to file or click the Save button to save the current pattern's data to a file.
