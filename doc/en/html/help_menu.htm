Help Menu

The Help Menu contains commands for getting help in using EPANET.

CommandDescription

Help TopicsDisplays the Help system's Help Topics dialog box

UnitsLists the units of measurement for all EPANET parameters

TutorialPresents a short tutorial introducing the user to EPANET

AboutLists information about the version of EPANET being used

Context-sensitive Help is also available by pressing the F1 key.
