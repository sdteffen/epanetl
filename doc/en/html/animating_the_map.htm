Animating the Map

The change in network parameters over time, as displayed in color-coded fashion on the Network Map, can be animated forward and backward in time. After selecting the parameters to view from the Map Browser, use the animation control push buttons on the Browser as follows:

Click to begin animating forward in time.

Click to begin animating backward in time.

Click to halt the animation.

Click to return to the initial time.

Use the slider control under the animation buttons to control the speed of the animation.

Only computed quantities, such as node pressure or link flow rate, can be animated. Design parameters, such as node elevation or link diameter, cannot be animated.
