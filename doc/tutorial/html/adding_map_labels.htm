Adding Map Labels

The final task in building our network is to add some descriptive labels.

1.Select the button on the Map Toolbar and click somewhere close to the reservoir (Node 1). An edit box will appear. Type in the word SOURCE and press the Enter key.

2.Click next to the pump and enter its label (PUMP), then do the same for the tank.

3.Click the button on the Toolbar to put the map into Object Selection mode rather than Text Insertion mode.
